# bee-progress-bar

记录进度或动态的显示进度变化.


## API

|参数|说明|类型|默认值|
|---|----|---|------|
|min|最小值|number|0|
|max|最大值|number|100|
|now|显示值|number |''|
|srOnly|label 只读不显示|bool|false|
|striped|条纹样式|bool|false|
|active|激活状态|bool|false|
|colors|颜色oneOf:danger,info,warning,success,primary,dark,''|string|''|
